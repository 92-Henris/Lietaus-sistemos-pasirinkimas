# Lietaus Sistemos Pasirinkimas

Šis projektas yra internetinė aplikacija lietaus sistemos (stogo nutekėjimo sistemos) pasirinkimui. Ji padeda vartotojams pasirinkti tinkamus stogo latakus, lietvamzdžius ir kitus elementus pagal jų poreikius.

## Funkcijos

- Interaktyvus pasirinkimas iš įvairių stogo elementų
- Vaizdinė pagalba su paveikslėliais
- Lengvai naudojama sąsaja

## Kaip naudoti

1. Atidarykite `index.html` failą naršyklėje.
2. Sekite instrukcijas ekrane, kad pasirinktumėte reikiamus elementus.

## Technologijos

- HTML
- CSS
- JavaScript

## Įdiegimas

Šis projektas yra statinis, todėl nereikia specialaus serverio. Tiesiog atidarykite `index.html` failą bet kurioje šiuolaikinėje naršyklėje.
